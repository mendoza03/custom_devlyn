# Odoo Biometric

Este scope agrupa los addons Odoo usados en el frente biométrico de Devlyn.

## Módulos incluidos

- `module/odoo_biometric_bridge/`
  Bridge de autenticación biométrica, auditoría y políticas de acceso.
- `module/hr_biometric_attendance_sync/`
  Sincronización de eventos biométricos hacia `hr.biometric.event` y `hr.attendance`.
- `module/devlyn_dahua_attendance_reporting/`
  Reporte de asistencias por sucursal con catálogos vivos y exportación XLSX.
