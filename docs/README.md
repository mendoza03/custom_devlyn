# Documentación

Este directorio contiene assets documentales vigentes para el frente biométrico.

## Contenido

- `catalogs/`
  Catálogos usados por el reporte de asistencias por sucursal.
- `reports/`
  Diagnosticos, runbooks y handoffs operativos del frente de reportes biometricos.
- `vendor/dahua/`
  Manuales del vendor Dahua usados como referencia técnica.

La documentación ejecutiva del stack biométrico vive en [`../BIOMETRIA.md`](../BIOMETRIA.md) y la del MCP server en [`../MCP.md`](../MCP.md).
